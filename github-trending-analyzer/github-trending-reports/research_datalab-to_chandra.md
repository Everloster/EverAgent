# datalab-to/chandra

> OCR model that handles complex tables, forms, handwriting with full layout.

## 项目概述

Chandra 是一个面向复杂文档 OCR 的模型与工具项目，强调的不只是“识别文字”，而是输出结构化 markdown / HTML / JSON，同时尽量保留原始布局、表格、表单和手写内容。相比传统 OCR 工具，它更接近“文档理解层”。

从 GitHub 仓库和 Hugging Face 模型卡来看，Chandra 的定位很明确：把复杂文档解析做成既可本地跑、又能接入服务端推理的现代 OCR 方案。它这次进入日榜，也说明“多模态文档理解”仍是高热度方向。

## 技术分析

### 技术栈

- **核心语言**：Python
- **模型形态**：OCR / VLM 风格文档解析模型
- **调用模式**：本地 Hugging Face 推理、vLLM 服务端推理
- **输出格式**：Markdown、HTML、JSON

### 架构设计

Chandra 的设计重点不在前端，而在“文档到结构化表示”的转换链：

- 输入 PDF / 图片
- 模型识别文本与布局
- 输出结构化文档表示
- 支持表格、表单、手写、图像说明等复杂要素

模型卡特别强调 40+ 语言支持、表格/数学/复杂布局处理，这表明它追求的是通用复杂文档理解，而非只做简单文本 OCR。

### 核心功能

- 文档转 Markdown / HTML / JSON
- 表格和表单重建
- 手写支持
- 图像与图表提取
- 本地与远程两种推理模式
