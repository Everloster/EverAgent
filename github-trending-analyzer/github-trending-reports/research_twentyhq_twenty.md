# twentyhq/twenty

> Building a modern alternative to Salesforce, powered by the community.

## 项目概述

Twenty 是一个现代化的开源 CRM 系统，旨在打造 Salesforce 的替代方案。与传统 CRM 不同，Twenty 强调开源、社区驱动和现代化技术栈，提供完整的客户关系管理能力，包括联系人管理、机会跟踪、任务管理等核心功能。2022 年上线，v1.19.0，41,146 Stars，是开源 CRM 领域最受关注的项目之一。

- **TypeScript 主导**：51.8M 行代码，现代 Web 开发标准
- **React + NestJS**：前后端分离架构
- **PostgreSQL**：可靠的关系型数据库
- **GraphQL**：灵活的 API 层
- **Monorepo**：统一的代码仓库管理

```

    ↓ GraphQL API
```
