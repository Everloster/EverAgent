# affaan-m/everything-claude-code

> The agent harness performance optimization system. Skills, instincts, memory, security, and research-first development for Claude Code, Codex, Opencode, Cursor and beyond.

## 项目概述

Everything Claude Code（ECC）是一个全面的代理工具链优化系统，由Affaan Mustafa经过10个月的日常使用打磨而成。该项目从个人配置包发展成为一个完整的"代理工具链性能优化系统"，适配Claude Code、OpenAI Codex、Cursor和OpenCode等多个AI编码平台。它通过组织化的技能、专家代理、记忆系统和严格的安全规则，为开发者提供了一个一站式的AI助手框架。项目目标是让开发者能够充分利用AI编码助手的潜力，实现更高效的研究优先开发和多语言支持。

## 技术分析

### 技术栈

Everything Claude Code采用多层次的技术架构，支持多种编程语言和框架生态：

**核心技术**：项目主要使用TypeScript、Node.js、Bash和Python实现，提供跨平台兼容性。TypeScript提供类型安全的组织方式，而Python用于数据处理和AI集成。

**支持的生态系统**：
- Web框架：Next.js、React、Vue.js等前端框架
- 后端框架：Django、Spring Boot、Laravel、Rust Axum、Go微服务
- 测试框架：Playwright（E2E测试）、pytest、GoogleTest
- DevOps工具：Docker容器化、PM2进程管理、CI/CD工作流
- 安全扫描：AgentShield集成、漏洞检测、代码审计

项目还支持7种主流编程语言：JavaScript、Python、Go、Java、Perl、C++和Rust，并在v1.9.0版本扩展至10个生态系统。

### 架构设计

Everything Claude Code采用模块化的多层架构设计，旨在为AI编码助手提供完整的工具链支持：

**组件层**：
- **28个专业代理（Agents）**：每个代理针对特定任务优化，包括规划代理、架构设计代理、代码审查代理、构建问题解决代理以及语言特定的代理
- **119+ 技能模块（Skills）**：包含TDD驱动开发、安全代码审查、前端模式、框架特定开发（Django/Spring/Laravel）、视频处理、内容创建等域工作流
- **60+ 斜杠命令（Commands）**：提供快捷操作如`/plan`（计划）、`/tdd`（测试驱动）、`/code-review`（代码审查）、`/build-fix`（构建修复）、`/learn`（学习）、`/evolve`（进化）
- **规则系统（Rules）**：按编程语言组织，包含TypeScript、Python、Go、Swift、PHP等语言的最佳实践
- **钩子系统（Hooks）**：支持会话生命周期自动化、记忆持久化、战略性压缩

**存储和状态管理**：v1.9.0引入了SQLite状态存储，支持跨会话的记忆持久化和智能压缩，使助手能够在多个对话中学习和改进。

**流程编排**：采用选择性安装架构，通过清单驱动的管道实现，支持循环防护，提高了大规模编排的可靠性。

### 核心功能

**代理工具链优化**：ECC通过组织专业代理，每个代理在特定领域（如架构设计、测试、安全审查）具有深度优化，使AI助手能够执行更精准的代码操作。

**多语言和框架支持**：支持全栈开发场景，从前端React/Vue到后端Django/Spring Boot，再到移动开发和微服务架构，覆盖现代开发的主要技术栈。

**安全扫描和合规**：集成AgentShield进行深度安全检查，包含102条安全规则，通过912个测试验证，覆盖OWASP Top 10漏洞类型、SQL注入检测、XSS防护、未验证输入检测等。

**智能学习和记忆系统**：通过SQLite存储跨会话数据，支持长期记忆持久化，使助手能够在不同的代码库和任务中积累经验。

**研究优先开发方法**：强调在编码前进行深度研究和规划，提供专门的研究代理和规划工具。

**E2E自动化测试**：集成Playwright框架，支持复杂的前端交互测试场景。

## 总结评价

### 优势

- **生态完整性**：提供了从规划、编码、测试到部署的全链路支持，覆盖现代开发的主要技术栈
- **安全性强**：集成AgentShield，包含102条安全规则，通过912个测试验证，覆盖OWASP漏洞
- **多平台适配**：支持Claude Code、Codex、Cursor等多个AI编码平台，避免厂商锁定
- **社区活跃**：日均新增star数超过4000，拥有30+贡献者，1282个测试确保质量
- **学习和进化**：内置记忆系统和学习代理，支持跨会话的知识积累和改进
- **开源且免费**：采用MIT协议，完全开源，降低使用成本

### 劣势

- **学习曲线陡峭**：大量的代理、技能、命令和规则需要时间理解和掌握
- **初学者友好性不足**：文档虽然丰富但可能过于详细，初学者容易被复杂性吓退
- **维护成本高**：28个代理、119+技能模块的维护需要社区的持续投入
- **性能开销**：完整的代理链、安全扫描和内存管理可能在资源受限环境中带来性能压力
- **定制化困难**：虽然声称支持定制，但实际定制化某些代理或技能可能需要深入的技术理解

### 适用场景

- **专业开发团队**：适合有技术深度的团队，需要确保代码质量和安全性
- **多语言全栈开发**：适合从前端到后端、支持多种框架的项目
- **安全性要求高的项目**：金融、医疗等领域，需要严格的安全审计
- **大规模AI辅助编码**：适合依赖AI助手进行大量代码生成的项目
- **开源项目管理**：适合需要统一的代码审查和质量保证流程的开源项目

**不适合的场景**：
- 小型个人项目或简单脚本开发
- 对学习成本敏感的初学者
- 资源受限的嵌入式或IoT项目
- 短期的一次性项目
