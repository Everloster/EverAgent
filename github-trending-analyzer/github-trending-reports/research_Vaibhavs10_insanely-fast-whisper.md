# Vaibhavs10/insanely-fast-whisper

> On-device Whisper transcription CLI

## 项目概述

insanely-fast-whisper 是一款本地 Whisper 转录 CLI 工具，专注于高速、高质量的语音转文字。与云端转录服务不同，insanely-fast-whisper 完全运行在本地设备上，支持 GPU 加速，实现极低的延迟和更好的隐私保护。用户可以在终端直接调用，无需网络连接即可完成音频转录。
