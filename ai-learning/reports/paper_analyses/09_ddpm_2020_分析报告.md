## 🎯 一句话总结

> DDPM 证明：将图像逐步加噪再学习逆向去噪，比 GAN 更稳定、比 VAE 更清晰，成为 Stable Diffusion / DALL·E 2 / Sora 的技术基石。

## 🌍 背景与动机（WHY）

### 问题：生成模型的三大困境

2020 年前，深度生成模型形成三大流派，各有致命缺陷：

| 方法 | 代表 | 优点 | 致命缺陷 |
|------|------|------|----------|
| **GAN（对抗训练）** | StyleGAN | 生成质量高 | 训练极不稳定（模式崩塌），需要精细调参 |
| **VAE（变分推断）** | β-VAE | 训练稳定 | 生成图像模糊，后验近似引入误差 |
| **Flow（标准化流）** | Glow | 精确似然 | 网络结构受限（必须可逆），参数量巨大 |

### 历史先驱：扩散模型的早期尝试

扩散模型并非 Ho 等人首创。Sohl-Dickstein et al. (2015) 首次将**热力学扩散过程**引入生成模型，提出"非平衡热力学"框架。但原始版本生成质量差，未引起广泛关注。

2019 年，Song & Ermon 提出**基于分数（score-based）的生成模型**，发现通过噪声扰动数据、学习分数函数（∇ log p(x)）可以生成高质量样本。

**Ho 等人的洞察**：
> "能否把扩散过程的理论框架与深度神经网络的强大表达能力结合，同时引入一系列工程细节（参数化方式、噪声调度、损失简化），让扩散模型在质量上与 GAN 抗衡？"

---

## 💡 核心贡献（WHAT）

1. **建立 DDPM 完整实用框架**：将前向扩散（加噪）与反向去噪（神经网络拟合）统一在概率框架内，并给出端到端训练和采样算法

2. **关键参数化简化**：证明训练目标可简化为**预测噪声 ε**（而非预测原始数据 x₀ 或均值 μ），大幅稳定训练

3. **扩散模型首次超越 GAN**：在 CIFAR-10 上以 **FID = 3.17** 超越所有先前非对抗生成模型，在 CelebA 256×256 上生成效果接近 StyleGAN

4. **揭示与去噪分数匹配的等价性**：连接了 DDPM 与 Song & Ermon 基于分数的方法，统一了两条研究线

## 🔧 技术方法（HOW）

### 核心思想：两个方向的马尔科夫链

```
前向过程（加噪，固定不训练）：
x₀（原图）→ x₁ → x₂ → ... → x_T（纯高斯噪声）
              每步加一点噪声

反向过程（去噪，神经网络学习）：
x_T（纯高斯噪声）→ ... → x₂ → x₁ → x₀（生成图像）
                   每步去一点噪声
```

### 前向过程：逐步加噪

给定原始图像 x₀，前向过程定义为：

```
q(x_t | x_{t-1}) = N(x_t; √(1-β_t) · x_{t-1}, β_t · I)
```

其中 β_t 是预设的噪声调度（noise schedule），从 β₁=0.0001 到 β_T=0.02 线性增大（T=1000步）。

**重要性质（闭合形式采样）**：

令 ᾱ_t = ∏(1-β_s)，则任意时刻的加噪结果可以直接从 x₀ 采样：

```
q(x_t | x₀) = N(x_t; √ᾱ_t · x₀, (1-ᾱ_t) · I)
即：x_t = √ᾱ_t · x₀ + √(1-ᾱ_t) · ε,  ε ~ N(0,I)
```

这个性质让训练无需真正执行 t 步加噪，可以直接跳到任意时刻！

### 反向过程：神经网络去噪

反向过程试图学习 p_θ(x_{t-1}|x_t)，即给定含噪图像 x_t，预测上一步 x_{t-1}：

```
p_θ(x_{t-1}|x_t) = N(x_{t-1}; μ_θ(x_t, t), Σ_θ(x_t, t))
```

Ho 等人的关键设计：**固定方差**（Σ_θ = σ²_t · I），只用神经网络预测均值 μ_θ。

### 参数化：预测噪声 ε（关键决策）

均值 μ_θ 可以用两种方式参数化：
- 预测 x₀（原始图像）
- 预测 ε（加进去的噪声）

Ho 等人发现**预测 ε 效果更好**，将 μ_θ 改写为：

```
μ_θ(x_t, t) = (1/√α_t) · (x_t - β_t/√(1-ᾱ_t) · ε_θ(x_t, t))
```

其中 ε_θ 是神经网络（U-Net 架构），输入当前含噪图像 x_t 和时间步 t，输出预测噪声。

### 训练目标简化

理论上应优化变分下界（ELBO），但 Ho 等人发现可以进一步简化为：

```
L_simple = E_{t,x₀,ε} [ ||ε - ε_θ(√ᾱ_t · x₀ + √(1-ᾱ_t) · ε, t)||² ]
```

**直觉**：随机选一张图像 x₀，随机选时间步 t，加入随机噪声 ε，训练网络预测这个噪声。就是这么简单！

### 采样算法

```python
# 训练（简化版）
for each training step:
    x₀ ~ data
    t ~ Uniform(1, T)
    ε ~ N(0, I)
    x_t = √ᾱ_t * x₀ + √(1-ᾱ_t) * ε
    loss = ||ε - ε_θ(x_t, t)||²
    update θ

# 采样（推断）
x_T ~ N(0, I)
for t = T, T-1, ..., 1:
    z ~ N(0, I)  if t > 1 else 0
    x_{t-1} = (1/√α_t) * (x_t - β_t/√(1-ᾱ_t) * ε_θ(x_t, t)) + σ_t * z
return x₀
```

### 网络架构：U-Net + 时间嵌入

神经网络 ε_θ 采用 **U-Net**（编码器-解码器 + 跳跃连接），额外引入：
- **时间步嵌入**（Sinusoidal Positional Encoding → MLP → 注入每层）
- **自注意力层**（在中间分辨率层加入）

## 📊 实验结果（RESULT）

### 定量指标

| 数据集 | 方法 | FID↓ | IS↑ |
|--------|------|------|-----|
| CIFAR-10 | DDPM（无条件） | **3.17** | **9.46** |
| CIFAR-10 | BigGAN（对比） | 14.73 | 9.22 |
| CIFAR-10 | NCSN（分数匹配） | 25.32 | 8.87 |
| CelebA 256×256 | DDPM | **5.11** | — |
| CelebA 256×256 | PGGAN（对比） | 8.00 | — |

**CIFAR-10 FID 3.17 首次让非对抗模型超越 GAN！**

### 定性观察

- 生成的 CIFAR-10 图像（32×32）在视觉上清晰自然
- CelebA 人脸生成质量接近专业 GAN
- **插值实验**：在噪声空间 x_T 做线性插值，中间图像过渡自然，说明隐空间有结构

### 主要局限

1. **采样速度慢**：需要 1000 步反向去噪才能生成一张图（后续 DDIM、DPM-Solver 等解决了此问题）
2. **分辨率限制**：原版 DDPM 主要在低分辨率（32×32、256×256）验证，高分辨率需要 Latent Diffusion（即 Stable Diffusion 的思路）
3. **无条件生成**：原版不支持文字/类别条件，后续 Classifier Guidance / CFG 解决

## 🧠 深度理解（INSIGHT）

### 为什么预测噪声比预测 x₀ 更好？

从信号处理角度：在高噪声时间步（t 接近 T），图像信息几乎消失，预测 x₀ 等于在噪声中猜答案。而预测噪声 ε 则是一个更稳定、更局部的任务——网络只需识别"加进去的是什么噪声"。

从梯度角度：预测 ε 的损失在各时间步上尺度更均匀，训练更稳定。

### 扩散模型 vs GAN 的本质区别

| 维度 | GAN | DDPM |
|------|-----|------|
| 训练稳定性 | ❌ 博弈不稳定，易崩塌 | ✅ 回归任务，MSE 损失 |
| 生成多样性 | ❌ 模式崩塌风险 | ✅ 随机采样，覆盖完整 |
| 采样速度 | ✅ 单次前向 | ❌ 需要 T=1000 步 |
| 理论基础 | 博弈论 + 近似 | 概率图模型 + 变分推断 |
| 可控生成 | 较难 | 较易（CFG 等方法） |

### 与分数匹配的统一

Song & Ermon 证明：**预测噪声 ε 等价于估计分数函数**（数据对数概率的梯度）：

```
ε_θ(x_t, t) ≈ -√(1-ᾱ_t) · ∇_{x_t} log p(x_t)
```

这意味着 DDPM 和基于分数的方法是同一事物的两种语言。Song et al. (2021) 在《Score-Based Generative Modeling through SDEs》中将两者统一为随机微分方程（SDE）框架。

## 📌 关键概念速查

| 概念 | 含义 |
|------|------|
| **扩散过程（Forward Process）** | 固定的马尔科夫链，逐步向数据加高斯噪声 |
| **反向过程（Reverse Process）** | 神经网络参数化的去噪马尔科夫链 |
| **噪声调度（Noise Schedule）** | β_t 序列，控制每步加噪强度 |
| **ᾱ_t（信号保留率）** | 原始信号在第 t 步剩余的比例，= ∏_{s=1}^{t}(1-β_s) |
| **分数函数（Score Function）** | ∇_x log p(x)，数据分布对数密度的梯度 |
| **FID（Fréchet Inception Distance）** | 评估生成质量的标准指标，越低越好 |
| **CFG（Classifier-Free Guidance）** | 后续工作，无需外部分类器实现条件生成 |
