## 第一网络：苏格拉底遗产（前470–前322）

### 核心谱系
```
苏格拉底（知识即德性，对话法）
├─→ 柏拉图（理念论）
│   └─→ 亚里士多德（形式内在论，批判理念）
└─→ 犬儒主义（否定约定俗成）
```

**关键张力**：柏拉图（超越主义，真理在理念世界）vs. 亚里士多德（经验主义，形式在具体事物）
- 这一分歧代表整个西方哲学的两种基本气质
- 文艺复兴名画《雅典学院》中，柏拉图手指向上，亚里士多德手掌朝下

**苏格拉底的悖论**：未写任何著作，却是西方哲学最重要源泉。通过柏拉图对话录与"苏格拉底方法"精神流传。

## 第二网络：近代哲学的双轨汇流

### 轨道A：大陆理性主义
```
笛卡尔（1596-1650）
  "我思故我在"，身心二元论，方法论怀疑
    │
    ├─→ 斯宾诺莎（1632-1677）
    │   批判：二元论内部矛盾
    │   替代：上帝即自然，一元论
    │
    └─→ 莱布尼茨（1646-1716）
        改造：无穷多精神单子（多元论）
        与牛顿争夺微积分发明权
```

### 轨道B：英国经验主义
```
洛克（1632-1704）：白板论、第一/第二性质、政治自由主义
    │
    ├─→ 贝克莱（1685-1753）
    │   激进化洛克：物质不存在，存在即被感知
    │
    └─→ 休谟（1711-1776）
        激进化到怀疑论：摧毁因果、实体、自我概念
        影响：惊醒康德，启发逻辑实证主义
```

### 汇流点：康德的综合
```
理性主义（笛卡尔）+ 经验主义（休谟的怀疑）
        ↓
康德（1724-1804）
"哥白尼式革命"：不是认识适应对象，而是对象适应认识
        ↓
开创德国观念论传统
```

## 第三网络：德国观念论与19世纪反叛

### 德国观念论谱系
```
康德 (先天综合判断，义务论)
  ↓ (激进化/批判)
费希特 (自我设定一切)
  ↓ (批判)
谢林 (自然哲学，早期浪漫主义)
  ↓ (批判)
黑格尔 (辩证法，绝对精神，历史哲学)
```

### 黑格尔的"后代"（思想继承）
```
黑格尔（1770-1831）
├─→ 马克思（1818-1883）
│   "将黑格尔倒转"：观念辩证法 → 历史唯物主义
│   "哲学家解释世界，关键是改变它"
│
└─→ 尼采的反叛
    叔本华（意志哲学，悲观主义） →
    尼采（超越叔本华，反黑格尔历史乐观主义）
      └─→ 海德格尔、福柯、德勒兹、存在主义
```

## 第四网络：20世纪的哲学分裂

### 分析哲学传统（英美）
```
弗雷格（1848-1925）：现代逻辑发明者
    ↓
罗素（1872-1970）：逻辑主义
    ↓
维特根斯坦（1889-1951）
  早期《逻辑哲学论》→ 维也纳学派 → 逻辑实证主义
  晚期《哲学研究》 → 日常语言哲学 → 牛津学派
  [早期与晚期几乎完全互相否定的罕见案例]
    ↓
奎因（1908-2000）：批判分析/综合之分，解体实证主义
    ↓
当代分析哲学各方向（本体论、认识论、心灵哲学）
```

### 欧陆现象学传统
```
胡塞尔（1859-1938）：现象学之父
    ↓ (师徒但哲学大转变)
海德格尔（1889-1976）：存在论
  ├─→ 萨特（1905-1980）：存在主义
  │   ├─→ 波伏娃（1908-1986）：存在主义女性主义
  │   └─→ 加缪（1913-1960）：荒诞主义
  │       [与萨特1952年决裂：历史必要性 vs. 道德绝对性]
  │
  ├─→ 伽达默尔（1900-2002）：诠释学
  └─→ 德里达（1930-2004）：解构主义
```

### 英美与欧陆的分裂
- 1950年代开始互不理解
- 分析哲学批评：欧陆晦涩、不精确
- 欧陆批评：分析哲学技术主义、脱离生活
- 1992年事件：剑桥授予德里达荣誉博士，18位哲学家（含奎因）联名反对

## 第五网络：政治哲学争论

```
洛克（自然权利、有限政府）
    ↓
边沁（功利主义）
    ↓
密尔（修正功利主义 + 自由主义）
    ↓
罗尔斯（1921-2002）：无知之幕，差别原则
    ↓ (直接对抗)
诺奇克（自由至上主义）vs. 桑德尔（社群主义）
```

**罗尔斯-诺奇克之争的本质**：
- 罗尔斯：正义关注最弱势者；国家有权再分配
- 诺奇克：权利不可侵犯；任何强制征税都是侵犯
- **背后哲学分歧**：人是社会存在 vs. 原子化个体
- **现实政治影响**：福利国家 vs. 市场自由主义的整个论辩根源

## 第六网络：心灵哲学与当代AI

```
笛卡尔的二元论遗产：心身如何交互？
    ↓ (现代版本)
意识如何从物质大脑中产生？
    │
    ├─→ 图灵（1912-1954）："机器能思考吗？"
    │
    ├─→ 塞尔（1932–）：中文房间——计算≠理解
    │
    ├─→ 丹尼特（1942–）：意识是幻觉（强物理主义）
    │
    └─→ 查默斯（1966–）：意识困难问题——物理主义不够
            ↓ (当代)
        LLM的"理解"与"意识"问题
```

**当代AI的哲学维度**（2020s最热）：
- 大型语言模型是"理解"还是"统计匹配"？（塞尔问题现代版）
- 若通过图灵测试是否有意识？（内格尔蝙蝠问题类比）
- 我们是否需要修改意识定义？（查默斯困难问题）

## 被低估的哲学家

- **玛丽·沃斯通克拉夫特**（1759-1797）：《女权辩护》，早于波伏娃150年
- **皮尔斯**（1839-1914）：美国实用主义之父，"有什么实际区别"判断真伪
- **弗雷格**（1848-1925）：现代逻辑奠基，被罗素发现悖论后精神崩溃

## 理解哲学传承的三个关键

1. **问题的持续性**：2500年一直在追问知识、自由、正义、意识——每代答案不同，问题相通
2. **批判的创造性**：几乎每个重要立场都通过批判前人而建立，批判不是否定，而是前进
3. **历史的不可逃避**：不理解笛卡尔，不能真正理解休谟；不理解休谟，不能理解康德
