```
    └────── virtue_ethics_eudaimonia          │
                       │                      │
            categorical_imperative            │
                       │                      │
            justice_as_fairness               │
                                              │
                                       (盖提尔反例诊断)
                       
                   dialectic
                  （独立的形而上学/方法论分支，
                    通过黑格尔→康德反向连接 categorical_imperative）
```

每条边的意义：
- socratic_method → theory_of_forms：定义追问预设统一本质 → 理念论
- socratic_method → jtb_knowledge：对"知识"的反复追问是 JTB 的远祖
- theory_of_forms → jtb_knowledge：理念论提供"真知识对象"的本体论基础
- cogito_ergo_sum → mind_body_dualism：Cogito 的"我"是 res cogitans，直接奠定二元论
- mind_body_dualism → qualia：内格尔是身心问题的现代化表达
- virtue_ethics ⟷ categorical_imperative：根本对照（品格 vs 规则、情境 vs 普遍）
- categorical_imperative → justice_as_fairness：罗尔斯继承康德人性公式作为原初状态前提
- jtb_knowledge → epistemic_luck：盖提尔反例的本质诊断

## 四、技术分歧与未决问题（截至 2026-04）

| 议题 | 立场 A | 立场 B |
|------|--------|--------|
| 形式存在 | 柏拉图：理念独立于事物之外 | 亚里士多德：形式内在于事物之中 |
| 知识来源 | 理性主义（笛卡尔/莱布尼茨）：先天观念 | 经验主义（洛克/休谟）：白板，从感觉来 |
| 道德根源 | 康德义务论：理性立法 + 普遍律 | 亚里士多德德性论：品格培养 + 实践智慧 |
| 政治正义 | 罗尔斯：差别原则 + 强制再分配 | 诺奇克：自由至上，权利不可侵犯 |
| 心灵本质 | 物理主义（丹尼特）：意识 = 脑活动 | 主观主义（内格尔/查默斯）：物理描述遗漏 qualia |
| JTB 修补 | 因果/可靠主义/美德认识论：分析性修补 | Williamson：知识不可分析，是原始概念 |
| 历史进程 | 黑格尔：辩证发展有方向 | 波普尔：开放社会，反历史决定论 |
