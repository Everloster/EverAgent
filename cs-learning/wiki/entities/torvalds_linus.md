# Linus Torvalds（1969– ）

## 身份
芬兰裔美籍软件工程师。Linux 内核与 Git 的创造者，Linux Foundation 核心维护者。来源：CS关键人物图谱 §第三代

## 核心贡献
- **Linux 内核（1991）**：21 岁时在赫尔辛基大学开始的"hobby"项目，现支撑全球 >95% 的云服务器。来源：CS关键人物图谱 §Torvalds
- **Git（2005）**：两周内写完初版的分布式版本控制系统，重塑现代开源协作范式。来源：CS关键人物图谱 §Torvalds

## 关键时刻
1991 年 Usenet 公告："I'm doing a (free) operating system (just a hobby, won't be big and professional like gnu)"。来源：CS关键人物图谱 §Torvalds

## 个人风格
以邮件列表上的直接批评著称；与 Tanenbaum 的 Linux vs. Minix 论战是 CS 历史上最著名的技术争论之一。来源：CS关键人物图谱 §Torvalds

## 传承影响
- Linux → Android / 服务器基础设施
- Git → GitHub → 现代开源协作
- 与 Thompson + Ritchie 的 Unix 谱系是最直接的传承（Linux 是 Unix-like 内核）
