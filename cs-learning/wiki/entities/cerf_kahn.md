# Vint Cerf & Bob Kahn

## 身份
TCP/IP 的共同发明者，互联网架构的奠基人。2004 年共同获图灵奖。来源：CS关键人物图谱 §图灵奖表

## 核心贡献
- **TCP/IP 架构（1974）**：在《A Protocol for Packet Network Intercommunication》中提出网关概念、端到端论点、五元组连接标识，奠定分组交换网络互联的基础。来源：19_tcpip_1974
- **Internet 开放治理**：参与 IETF / ICANN，推动互联网协议作为开放标准而非专有技术。来源：CS关键人物图谱

TCP/IP → 现代互联网 → 所有分布式系统的底层（GFS / MapReduce / Kafka / Spanner 全部依赖之）
