# C.A.R. Hoare（Tony Hoare，1934– ）

## 身份
英国计算机科学家。牛津大学、微软研究院剑桥。1980 年图灵奖得主。来源：CS关键人物图谱 §第二代

## 核心贡献
- **快速排序（Quicksort，1960）**：至今最常用的排序算法之一。来源：CS关键人物图谱 §Hoare
- **Hoare Logic（1969）**：用前置/后置条件做程序正确性的形式化验证。来源：CS关键人物图谱 §Hoare
- **CSP（通信顺序进程，1978）**：以"通信代替共享内存"的并发代数理论，直接催生 Go 的 goroutine + channel。来源：18_csp_1978
- **空指针**：发明 null reference，他自己称之为"十亿美元的错误"（2009 年自我批评）。来源：CS关键人物图谱 §Hoare
