# Ken Thompson（1943– ）& Dennis Ritchie（1941–2011）

## 身份
贝尔实验室计算科学研究中心的合作搭档。Unix 与 C 语言的共同创造者。1983 年共同获图灵奖。Thompson 后加入 Google，参与 Go 语言设计。来源：CS关键人物图谱 §第二代

## 核心贡献（合作）
- **Unix 操作系统（1969–1974）**：奠定"一切皆文件"、管道、shell 等系统设计哲学。来源：04_unix_1974
- **B → C 语言（Ritchie 主导，1972）**：可移植的系统编程语言，重写 Unix 内核证明高级语言写 OS 的可行性。来源：04_unix_1974

## 个人贡献区分
- **Thompson**：Unix 核心设计、B 语言、Go 语言（Google，2007，与 Pike、Griesemer 合作）、UTF-8 编码（与 Pike 合作）。来源：CS关键人物图谱 §Thompson_Ritchie
- **Ritchie**：C 语言完整设计、《The C Programming Language》教材（与 Kernighan 合作）。来源：CS关键人物图谱 §Thompson_Ritchie

## 历史注脚
Ritchie 于 2011 年去世，仅在 Steve Jobs 去世后一周，但媒体报道远不及 Jobs。Ars Technica 称其为"创造了现代计算的男人"。来源：CS关键人物图谱 §Thompson_Ritchie

- Unix → Linux → macOS → Android（所有主要 OS 的直接祖先）
- C → C++ / Java / 大多数现代语言
- Thompson 后续：Go 的 channel 设计承袭 Hoare 的 CSP
