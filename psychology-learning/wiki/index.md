## 使用指南

1. 查找某位心理学家 → 先读 `entities/`
2. 理解某个机制 → 先读 `concepts/`，每页按 L1–L5 五层展开
3. 跨实验 / 跨概念综合 → 关注 `concepts/situationism.md`、`concepts/heuristics_and_biases.md`
4. 深度 ≥ 3 概念合成查询 → 结果归档到 `syntheses/`
