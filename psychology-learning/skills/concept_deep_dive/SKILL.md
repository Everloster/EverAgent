# 技能：心理学概念深度解析

> 基于通用模板：[SKILL_TEMPLATES.md](../../docs/SKILL_TEMPLATES.md) · 模板 B
> 本文件为领域特化版本，增加了以下专属模块：经典实验回顾、跨文化适用性、现实行为启示

## 技能描述

当用户想要深入理解某个心理学概念、理论框架或心理现象时使用此技能。提供从直觉类比到学术分析的分层解析，帮助学习者真正理解而不仅仅是记忆。

**触发场景**：
- "深入讲解'认知失调'是什么"
- "帮我理解双系统理论"
- "详细解释习得性无助"
- "概念深挖：[主题]"
- "从直觉到理论讲解[心理学概念]"
- "这个心理现象是怎么回事？"

## 使用说明

**执行步骤**：
1. 识别用户要学习的心理学概念
2. 使用 WebSearch 搜索 Simply Psychology、APA Dictionary、Wikipedia 英文版等资源
3. 按上述框架从直觉到专业逐层解析
4. 生成完整的知识解析报告（MD 格式）
5. 保存到 `reports/concept_reports/` 目录
6. 提供文件链接给用户

**输出文件命名**：
`[概念名称]_深度解析.md`
例如：`认知失调_深度解析.md`、`习得性无助_深度解析.md`

## 核心心理学概念快速索引

以下是最值得用此技能深度解析的心理学概念：

### 认知心理学核心概念
- `工作记忆（Working Memory）` — 信息加工的核心瓶颈
- `认知负荷（Cognitive Load）` — 学习效率的根本制约
- `注意（Attention）` — 选择性注意、持续注意、分配注意
- `组块（Chunking）` — 突破工作记忆限制的关键策略
- `图式（Schema）` — 知识组织的基本单元
- `间隔效应（Spacing Effect）` — 为什么间隔重复比集中学习更有效

### 社会心理学核心概念
- `认知失调（Cognitive Dissonance）` — 信念与行为的矛盾
- `旁观者效应（Bystander Effect）` — 责任扩散的机制
- `从众（Conformity）` — 规范性 vs. 信息性社会影响
- `服从（Obedience）` — 权威的力量与情境因素
- `情境力量（Situational Forces）` — 环境对行为的决定性影响
- `基本归因错误（FAE）` — 高估人格、低估情境的倾向
- `自我服务偏差（Self-Serving Bias）` — 成功归功于自己、失败归咎于环境

### 行为经济学核心概念
- `损失厌恶（Loss Aversion）` — 损失的感受比等量收益更强烈
- `锚定效应（Anchoring Effect）` — 初始信息对判断的不成比例影响
- `可得性启发（Availability Heuristic）` — 容易回忆的=更可能的
- `代表性启发（Representativeness Heuristic）` — 用相似性代替概率判断
- `框架效应（Framing Effect）` — 同一信息的不同表述改变选择
- `双系统理论（Dual Process Theory）` — 系统1 vs. 系统2

### 动机与情绪
- `内在动机 vs. 外在动机` — 过度辩护效应的核心
- `习得性无助（Learned Helplessness）` — 无法控制感对主动性的摧毁
- `自我效能感（Self-Efficacy）` — 能力信念对行为的影响
- `心流（Flow）` — 最优体验的条件与特征
- `情绪调节（Emotion Regulation）` — 重评与压制的对比
- `情绪劳动（Emotional Labor）` — 职业情绪管理的代价

### 发展心理学核心概念
- `依恋类型（Attachment Styles）` — 安全型 / 焦虑型 / 回避型
- `近端发展区（ZPD）` — 维果斯基的学习理论核心
- `同化与顺应（Assimilation/Accommodation）` — 皮亚杰认知发展机制
- `道德推理发展（Moral Development）` — 科尔伯格的三水平六阶段

### 临床/人格心理学核心概念
- `防御机制（Defense Mechanisms）` — 压抑、投射、合理化、升华
- `认知扭曲（Cognitive Distortions）` — CBT 核心：全或无思维、灾难化等
- `大五人格（Big Five / OCEAN）` — 现代人格心理学的标准框架
- `创伤后应激障碍（PTSD）` — 创伤记忆的神经心理机制
- `正念（Mindfulness）` — 当代临床心理学的核心干预工具

### 神经科学与心理
- `躯体标记假说（Somatic Marker Hypothesis）` — 情绪是决策的向导
- `神经可塑性（Neuroplasticity）` — 大脑改变的能力与机制
- `杏仁核劫持（Amygdala Hijack）` — 恐惧如何绕过理性思维
- `镜像神经元（Mirror Neurons）` — 模仿与同理心的神经基础（尚有争议）
