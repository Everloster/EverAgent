## 项目目标

建立一套完整的心理学自学体系，覆盖近150年心理学发展史、核心实验精读、关键理论研究，并持续积累深度分析报告。
项目的三个维度：**实验深度**（经典研究精读）× **历史叙事**（人物图谱、范式革命）× **当代相关性**（心理学如何照亮 AI、行为经济学、自我认知）。

## 🗓️ 推荐学习节奏

```
Week 1-2:   心理学发展时间线 + 人物图谱（建立历史视野，理解范式演替）
Week 3-4:   Miller (1956) 认知革命起点 + Milgram (1963) 服从权威实验
Week 5-6:   Festinger (1959) 认知失调 + Harlow (1958) 依恋与爱的实验
Week 7-8:   Seligman (1967) 习得性无助 + Bandura (1961) 社会学习理论
Week 9-10:  Kahneman & Tversky (1974, 1979) 双系统思维 + 前景理论
Week 11-12: Rosenhan (1973) + Zimbardo (1971) 斯坦福监狱实验（伦理与情境力量）
Week 13-14: 弗洛伊德精神分析 + 荣格原型理论（人格理论专题）
Week 15-16: Maslow 需求层次 + Rogers 人本主义 + Seligman 积极心理学
Week 17-18: 认知神经科学入门（Damasio, LeDoux）
Week 19+:   选择感兴趣方向深入（神经科学 / 临床心理学 / 行为经济学 / 进化心理学）
```

## 📌 三个核心视角（项目特色）

本项目在标准心理学学习之外，额外引入三个常被忽视的视角：

**1. 实验范式视角**：心理学的每次革命，都是一次测量工具的革命。
→ 从内省法（冯特）→ 行为观察（华生）→ 反应时（认知心理学）→ fMRI（神经科学），见 [心理学发展时间线](./roadmap/Psychology_Development_Timeline.md)

**2. 人的视角**：弗洛伊德与荣格为何决裂？斯金纳与乔姆斯基之争意味着什么？卡尼曼如何拿了诺贝尔经济学奖？
→ 见 [心理学关键人物图谱](./reports/concept_reports/心理学关键人物图谱.md)

**3. 当代相关性视角**：古老的心理学实验如何照亮今天的 AI 对齐、行为设计、精神健康困境？
→ 贯穿整个项目，每份报告末尾均有"当代启示"章节

## 🏷️ 心理学分支索引

| 分支             | 核心问题            | 代表作                                  |
| -------------- | --------------- | ------------------------------------ |
| **认知心理学**      | 人如何获取、存储、加工信息？  | Neisser《认知心理学》、Kahneman《快思慢想》        |
| **发展心理学**      | 人的心理如何随年龄变化？    | Piaget《智力心理学》、Erikson《童年与社会》         |
| **社会心理学**      | 他人如何影响个体的思想与行为？ | Milgram《服从权威》、Zimbardo《路西法效应》        |
| **人格心理学**      | 人与人之间的心理差异从何而来？ | Freud《自我与本我》、Jung《人及其象征》             |
| **临床/异常心理学**   | 心理障碍如何定义、诊断与治疗？ | Rosenhan (1973)、Beck《抑郁的认知疗法》        |
| **神经科学/生理心理学** | 大脑如何产生心理与行为？    | Damasio《笛卡尔的错误》、LeDoux《情绪大脑》         |
| **积极心理学**      | 什么让人类繁荣与幸福？     | Seligman《真实的幸福》、Csikszentmihalyi《心流》 |
| **行为经济学**      | 人类决策中有哪些系统性偏差？  | Kahneman & Tversky 系列论文              |
