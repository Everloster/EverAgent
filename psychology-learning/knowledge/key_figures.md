# 心理学关键人物：150年思想传承

## 奠基者（1870s–1910s）

### Wilhelm Wundt (1832–1920)
- **核心贡献**：建立世界第一个心理学实验室（1879，莱比锡）；创立构造主义；心理学从哲学独立
- **方法论遗产**：内省法（后被批判，但"心理学必须是实验科学"的信念塑造学科）
- **学生网络**：铁钦纳（美国）、霍尔（创立APA）、卡特尔（心理测量先驱）
- **局限**：内省的主观性导致"不同实验室互相矛盾"，直接催生行为主义革命

### William James (1842–1910)
- **核心贡献**：《心理学原理》（1890）；功能主义；意识流；詹姆斯-兰格情绪理论
- **关键洞察**：心理不研究"结构是什么"，而研究"功能是什么"（为进化心理学埋伏笔）
- **评价冯特**："无聊至极的事实堆砌"（代表两大派系的对立）
- **传承**：实用主义哲学→行为主义实践取向→人本主义

### Sigmund Freud (1856–1939)
- **革命性洞察**："最重要的心理活动发生在你看不见的地方"（无意识理论）
- **核心贡献**：精神分析；防御机制；本我/自我/超我；梦的解析；童年决定论
- **关键决裂**：阿德勒（强调社会因素）、荣格（拒绝泛性论，集体无意识）
- **当代遗产**：防御机制被现代认知科学部分吸收；依恋理论受影响；但多数具体主张缺乏实证支持

## 行为主义革命（1910s–1950s）

### John B. Watson (1878–1958)
- **宣言**：《心理学应该是一门自然科学》（1913）
- **激进主张**："给我任意一打健康婴儿……不论才能祖先，我能训练成任何专家"
- **影响**：极端后天论；广告心理学先驱（离开学术界后）

### Ivan Pavlov (1849–1936)
- **发现**：经典条件作用（意外发现，研究消化系统时）
- **贡献**：为行为主义提供生理学证据
- **背景**：俄国生理学家，思想传统与华生不同但被借用

### B.F. Skinner (1904–1990)
- **核心贡献**：操作条件作用；斯金纳箱；强化程序；行为塑造；《超越自由与尊严》
- **与华生区别**：华生研究被动的经典条件作用，斯金纳研究主动的操作条件作用
- **关键论战**（1959）：
  ```
  斯金纳《言语行为》(1957)：语言通过强化习得
        ↓
  乔姆斯基长篇书评批判：儿童产生从未听过的句子；
              语言有"贫乏刺激"问题→必有先天语法装置
        ↓
  结果：认知革命的标志性事件；行为主义开始衰退
  ```

## 认知革命与社会心理学黄金期（1950s–1980s）

### George A. Miller (1920–2012)
- **核心贡献**：《神奇的数字七》（1956）；工作记忆容量限制；认知科学奠基
- **历史地位**：信息论引入心理学，标志认知革命起点
- **影响力**：培育了认知科学家一代

### Jean Piaget (1896–1980)
- **核心贡献**：认知发展四阶段理论；图式；同化与顺应
- **独特性**：在日内瓦独立于美国主流（直到1960年代才被广泛接受）
- **继承者**：维果斯基（社会文化因素）；新皮亚杰主义（信息加工视角）

### John Bowlby (1907–1990)
- **核心贡献**：依恋理论；母婴分离的长期影响
- **知识背景**：精神分析训练+动物行为学（劳伦兹印刻研究）
- **对精神分析的批判**：婴儿依恋母亲不是因为食物，而是独立的进化机制
- **证实者**：哈洛猴子实验（1958）
- **继承者**：玛丽·安斯沃斯（陌生情境、依恋类型）；玛丽·梅因（成人依恋）

### Stanley Milgram (1933–1984)
- **核心贡献**：服从权威实验系列（1963-1974）；六度分隔理论
- **导师关系**：师从所罗门·阿什（从众实验）→ 将从众推向极端的服从研究
- **历史影响**：直接推动APA伦理准则（1973）制定

### Leon Festinger (1919–1989)
- **核心贡献**：认知失调理论（1957）；社会比较理论
- **著名故事**：潜入末世邪教研究——预言失败后信徒反而更坚定，启发认知失调理论

### Philip Zimbardo (1933–2024)
- **核心贡献**：斯坦福监狱实验（1971）；情境力量理论；《路西法效应》
- **与Milgram对比**：权威服从 vs. 角色与情境力量；单次任务 vs. 6天持续沉浸
- **晚年转向**：从"什么让好人变坏"→"什么让普通人成为英雄"

## 认知神经科学与行为经济学（1970s–2010s）

### Daniel Kahneman (1934–2024) & Amos Tversky (1937–1996)

**心理学史上最重要的学术合作之一**

```
卡尼曼（直觉与认知） + 特沃斯基（数学与逻辑）
        ↓ (1971-1996：共同发表改变经济学与心理学的论文)
启发式与偏差（1974）+ 前景理论（1979）+ 框架效应
        ↓
1996：特沃斯基因癌症逝（59岁）
2002：卡尼曼获诺贝尔经济学奖（特无法分享）
2011：卡尼曼《快思慢想》——双系统理论总结
2024：卡尼曼逝世（90岁）
```

**特沃斯基的遗言**："他是我见过最聪明的人"（卡尼曼自述获奖最悲伤的是无法与他分享）

**影响**：系统性证明"理性经济人"假设错误，直接催生行为经济学学科。

### Albert Bandura (1925–2021)
- **核心贡献**：社会学习理论；自我效能感；社会认知理论
- **从行为主义到认知心理学的桥梁**：
  ```
  传统行为主义：直接强化才能改变行为
        ↓
  班杜拉（Bobo娃娃实验，1961）：观察他人也能学习
        ↓
  人是认知主体，非被动刺激-反应机器
  ```
- **自我效能感**：不是"我有没有能力"，而是"我相信自己有能力吗"
  - 影响选择、努力程度、坚持时间
  - 动机心理学与认知心理学的交汇

### Martin E.P. Seligman (1942–)
**职业轨迹的弧线**：习得性无助（1967）→ 抑郁认知模型 → 积极心理学创始人（1998）

```
同样经历无助环境，为什么有些人崩溃，有些人反弹？
        ↓
研究解释风格：悲观者将失败归为持续、普遍、内部原因
        ↓
1998年APA主席演讲："什么让人幸福"
积极心理学运动正式启动
        ↓
PERMA模型；《蓬勃》（2011）
```

### Antonio Damasio (1944–)
- **核心贡献**：躯体标记假说；《笛卡尔的错误》（1994）；情绪作为认知基础
- **经典病例**：菲尼亚斯·盖奇（1848，脑部铁棒贯穿）
  - 身体存活，人格与决策能力完全改变
  - 证明前额叶-情绪-决策的关系
- **哲学对话**：笛卡尔预设理性与情绪分离——达马西奥用神经科学否定

## 思想传承脉络简图

```
冯特(实验实证) ──→ 构造主义(铁钦纳)
        ↓ (批判)
詹姆斯(功能主义)

华生(行为主义) → 斯金纳(操作条件作用)
        ↓ (乔姆斯基批判，1959)

皮亚杰(独立) → 认知发展理论
鲍尔比(进化论) → 依恋理论 → 安斯沃斯/梅因

弗洛伊德(精神分析)
        ├─ 阿德勒(社会因素)
        └─ 荣格(集体无意识)

米尔格拉姆(服从) ← 阿什(从众)
津巴多(情境力量)

卡尼曼+特沃斯基(决策) → 行为经济学
班杜拉(社会学习)
塞利格曼(习得性无助 → 积极心理学)
达马西奥(情绪与决策)
```

|--|---------|---------|
| 研究对象 | 可观察行为 | 内部心理过程 |
| 核心比喻 | 反射弧 | 计算机 |
| 代表人物 | 华生、斯金纳 | 米勒、奈瑟尔 |
| 结果 | 1960年代后衰退 | 成为现代主流 |
