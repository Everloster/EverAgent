## 项目目标

建立一套完整的生命科学自学体系，聚焦于**时间生物学（Chronobiology）× 睡眠科学（Sleep Science）× 运动生理学（Exercise Physiology）× 神经科学（Neuroscience）** 四个核心分支，并持续积累深度研究报告。

项目的三个维度：**机制深度**（生理机制精读）× **历史叙事**（人物图谱、范式革命）× **当代相关性**（生命科学如何指导个人健康、运动表现与认知优化）。

## 📌 三个核心视角（项目特色）

本项目在标准生命科学学习之外，额外引入三个视角：

**1. 机制优先视角**：不只问"什么有效"，而是问"为什么有效"——理解激素级联、信号通路、基因表达，才能将知识迁移到新情境。

**2. 时间维度视角**：生命现象都有时间结构。同一种干预，在不同时间点效果完全不同——这是时间生物学最深刻的洞见。

**3. 个体化视角**：人与人在基因层面的节律差异是真实存在的。优化生物学状态，必须从了解自己的昼夜节律类型开始，而不是套用群体平均数据。
