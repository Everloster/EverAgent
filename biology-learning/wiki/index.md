## 使用指南

1. 查找某位研究者 → 先读 `entities/`
2. 理解某个机制 → 先读 `concepts/`，每页按 L1–L5 五层展开
3. 跨主题综合（如"晚型人训练时间"）→ 同时读 `social_jetlag` + `chronotype` + `gh_sleep_coupling` + `sleep_architecture`
4. 深度 ≥ 3 概念合成查询 → 结果归档到 `syntheses/`
